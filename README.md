# India Village API Dashboard

Professional frontend dashboard for the India Village API project.

## Features

- React + Vite frontend
- Live backend connection
- Village search system
- API dashboard UI
- Backend status checker
- Professional dark theme

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Node.js
- Express.js
- PostgreSQL
- Prisma

## Backend API

Backend runs on:

```bash
http://localhost:3000
```

Frontend runs on:

```bash
http://localhost:5173
```

## API Endpoints

### Health Check

```bash
GET /health
```

### Get Districts

```bash
GET /districts
```

### Get Subdistricts

```bash
GET /subdistricts
```

### Get Villages

```bash
GET /villages
```

### Village Search

```bash
GET /autocomplete
```

### Register API Client

```bash
POST /register
```
